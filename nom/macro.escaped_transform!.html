escaped_transform!(&[T] -> IResult<&[T], &[T]>, T, &[T] -> IResult<&[T], &[T]>) => &[T] -> IResult<&[T], Vec<T>> matches a byte string with escaped characters.

The first argument parses the normal characters, the second argument is the control character (like \ in most languages), the third argument matches the escaped characters and trnasforms them.

As an example, the chain abc\tdef could be abc def (it also consumes the control character)
